HTML & CSS Basics Summary

HTML Summary

HTML(HyperText Markup Language) is used to define our page content,structure and meaning.You don't use it for styling purposes.Use CSS for that instead!

  • HTML uses "elements" to describe(annotate) content
  • HTML elements typically have an opening tag,content and then a closing tag
  • You can also have void(empty) elements like images
  • You can also configure elements with attributes
  • There is a long list of available elements but you will gain experience over time.

Learn more about all available HTMl elements on the MDN HTML elements reference

CSS Summary

CSS(Cascading Style Sheets) is used for styling your page content.

  • Styles are assigned via property-value pairs
  • You can assign styles via "style" attribute
  • To avoid code duplication, you typically use global styles(e.g via the 'style' element) instead
  • Alternatively, you can work with external stylesheet that you can link
  • When working with CSS,concepts like "inheritance","speciality" and the "box model" should be understood

Learn more about all CSS properties value on the MDN CSS property reference
